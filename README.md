
# IBP LatAm – El Impostor (PWA) · v4

## Cambios de esta versión
- Temas limitados a 10 palabras: Mate, Sushi, Harry Potter, Fast & Furious, Los Simpson, Johnny Depp, George Clooney, Extraterrestre, Paris, New York.
- Sin temas vagos para impostor (tema vacío).
- Regla de múltiples impostores: 1 impostor cada 4 jugadores (mínimo 1).
- Puntaje: acertar = +1 a cada tripulante; fallar = +2 a cada impostor.
- Compatibilidad GitHub Pages (service worker y rutas relativas).

## Publicación
1) Subí el contenido de esta carpeta a la **raíz** de tu repo.
2) Activá **GitHub Pages** → Source: Deploy from a branch → Branch: `main` / `/root`.
3) Esperá 1–2 min y abrí la URL `https://usuario.github.io/tu-repo/`.

## Uso rápido
1) Crear sala (modo Links, hasta 30 jugadores).
2) Asignar roles y tema → Copiar links privados.
3) Timer → Votación → Calcular acusado → Puntajes.

> Si no ves cambios: **Hard Reload** (Ctrl+F5 / Cmd+Shift+R) para limpiar cache del service worker.
