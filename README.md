
# IBP LatAm – El Impostor (PWA, sin backend)

## Novedades de esta versión
- Máximo **30 jugadores**.
- Modo **Links** seleccionado por defecto.
- **Branding** configurable vía `brand.json` (colores, nombre y logo).

## Deployment (HTTPS)
Subí estos archivos a GitHub Pages, Netlify, Vercel o SharePoint estático. La PWA requiere HTTPS para instalarse como app.
